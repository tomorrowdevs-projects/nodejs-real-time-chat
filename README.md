# Create a Real Time chat using NodeJS and Socket.io

## Requirements:
- Implement a local web server using NodeJs and Socket.io
- Implement an interface to show the real-time chat
- Each user should be authenticated on a database before joining the chat 
- Test the application from a different browser screen
